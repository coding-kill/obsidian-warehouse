## 5.2 角色成长域

| 表名 | 用途 |
|------|------|
| CharacterBasicTable | 角色基础属性（模板） |

## 5.3 技能战斗域

### Stance + Skill + Ability 三层结构

```
┌──────────────────────────────────────────────────────────────┐
│                    技能系统层次结构                           │
└──────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│  StanceBasicTable (架势层)                                   │
│  ├── Skill[0] ──► SkillBasicTable ──► SkillLevelTable     │
│  ├── Skill[1] ──► SkillAbilityTable ──► SkillAbilityValueTable│
│  └── Skill[2] ──► CooltimeManager                          │
└─────────────────────────────────────────────────────────────┘

玩家释放技能链路：
客户端施法 ──► CharacterSkillManager ──► getCurrStacnceSkill()
          ──► 检查stance/skill等级/冷却 ──► 执行技能效果
```

### 核心表

| 表名 | 用途 |
|------|------|
| SkillBasicTable | 技能基础属性 |
| SkillLevelTable | 技能等级配置 |
| SkillAbilityTable | 技能效果定义 |
| SkillAbilityValueTable | 技能效果数值 |
| SkillUpgradeTable | 技能升级配置 |
| StanceBasicTable | 架势定义（挂载技能） |

### 常见报错

| 报错 | 原因 |
|------|------|
| `not found curr Stance` | 角色当前stance无效 |
| `not curr stance skill` | stance里没有该技能 |

## 5.5 地图与场景域（MMO特色）

### Terrain 核心概念

| 概念 | 含义 |
|------|------|
| terrain_idx | 地图模板编号（内容设计维度） |
| terrain_uid | 运行时地图实例（服务器执行维度） |
| terrain_group_id | 地图分片归属（部署扩容维度） |
| GAME/INST/WAR | 不同类型地图承载服务 |

### 关键代码

```
TerrainManager.h:
  bool initTerrains();
  Terrain* getTerrain(TERRAIN_UID _terrainUid);
  Terrain* getTerrainForInstance(TERRAIN_IDX _terrainIdx);
  bool allSpawnMonster();
```

## 5.9 GameServer启动时的系统初始化顺序

```
GameServer.cpp -> startingService() 核心顺序：

1. 基础配置与鉴权
   ├── CONFIG_PTR->LoadConfig(...)
   └── AuthenticationSrvManager::Authentication()

2. 数据与基础模块
   ├── DataTable 初始化
   ├── Send 模块初始化
   ├── Terrain 初始化
   ├── 对象池初始化
   ├── Packet pool 初始化
   └── XignCode 初始化（可选）

3. 会话/监听端口
   ├── SESSION_MGR / USER_MGR 初始化
   ├── 客户端 IOCP（主端口）
   └── Arena/AdminTool/Billing 对外监听

4. 后端服务连接
   ├── DBAgent
   ├── Redis
   ├── Actor/怪物生成
   ├── 业务管理器初始化(Quest/Item/Guild...)
   ├── LogServer
   ├── WorkThreadManager
   ├── WorldServer
   ├── CommunityServer
   ├── InstancePoolServer（仅Instance服）
   ├── ChatServer
   ├── AuctionServer（仅Game服）
   ├── BillingServer
   └── ArenaServer（仅Game服）

5. 启动后首轮业务同步
   ├── TERRAIN_MGR->send_loadDB()
   ├── EventMonsterDropManager::send_LoadDB()
   └── BARON_RANKING_MNG->loadData()
```
