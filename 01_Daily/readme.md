# 每日笔记

日记、日复盘、临时待办。

## 目录结构

```
01_Daily/
├── logs/              # 随手记录碎片（原始）
├── daily/             # 正式日报（按年月 YYYY-MM/ 分类）
│   ├── 2025-05/
│   └── 2025-06/
├── Templates/         # 模板库
│   ├── 日报模板.md
│   └── 随手记模板.md
├── 计划总览.md         # 计划文件
└── readme.md          # 本说明
```

## 工作流程

### 随手记录
1. 打开或创建今日随手记：`logs/YYYY-MM-DD.md`
2. 随时记录工作内容、想法、待办
3. 格式参考 `logs/随手记模板.md`

### 整理日报
1. 第二天打开昨日随手记
2. 根据内容填写正式日报模板
3. 保存到 `daily/YYYY-MM/YYYY-MM-DD.md`
4. 清空或归档随手记

### 追加计划
1. 打开 `计划总览.md`
2. 根据性质添加到对应区域
3. 定期回顾和整理计划
